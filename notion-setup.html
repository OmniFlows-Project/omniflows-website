OmniFlows

Notion Integration Guide

Connect OmniFlows to your Notion workspace to sync knowledge seamlessly.

1
Create an Integration

You need an "Internal Integration" to allow the extension to access your Notion pages.

  1. Go to notion.so/my-integrations.
  2. Click + New integration.
  3. Give it a name like OmniFlows and click Submit.
  4. Copy the Internal Integration Secret. This is your API Token.

2
Prepare the Page & Connection

Notion requires you to explicitly grant an integration access to a page.

  1. Create a new Page in Notion (e.g., "My Research").
  2. Click the **...** (three dots) in the top right corner of the page.
  3. Select Add connections and search for the OmniFlows integration you just created.
  4. Confirm the connection.

3
Get the Page ID

OmniFlows needs to know where to create your research database.

  1. Copy the URL of the page you just created.
  2. The Page ID is the 32-character string at the end of the URL.

Example URL:

https://www.notion.so/My-Page-347eb299574480fe9d9bdcaf8cf9be41

The ID is 347eb299574480fe9d9bdcaf8cf9be41.

4
Finalize in OmniFlows

In OmniFlows Options, paste your Integration Secret and Page ID.

Automatic Setup

OmniFlows will automatically create a structured database (with tags, properties, and summaries) on that page when you save your first capture.

!
Troubleshooting

"Object not found"

Ensure you have completed Step 2. Even if you have the correct API Key, you must manually "Add Connection" to the specific page in Notion.

Database ID vs Page ID: If you are connecting to an existing database, use its ID. If connecting to a page, use the Page ID and let OmniFlows create the database for you.
